# Utilities

This section contains utility modules provided by the Spotify API library.  
All functions are available via their respective namespaces:

- `Spotify::Parse` – Parsing utilities for Spotify IDs and URIs.
- `Tools::` – General helper functions such as environment loading and time formatting.

---

## Modules

| Module        | Description                                                                           | Link                                             |
|---------------|---------------------------------------------------------------------------------------|--------------------------------------------------|
| **Parsing**   | Extract and validate Spotify IDs and URIs from URLs or URIs.                          | [View Documentation](/docs/Utilities/Parsing.md) |
| **Tools**     | General utility functions (environment, market validation, timestamps, formatting).   | [View Documentation](/docs/Utilities/Tools.md)   |

---

## Notes

- Parsing functions include **throwing** and **non-throwing (`try*`)** variants.
- Tools functions simplify common tasks like environment setup, timestamp generation, and duration formatting.
- Use this index to navigate to the detailed documentation for each module.
